# Implementasjon oav OAuth Code Flow
Autorisasjons kode (authorization code) er en midlertidig kode som klient kan bytte inn mot et access token.
Koden er levert av FakeIdentityAndAccess. Bruker for da en mulighet til å gi godkjenning til at WebBFFAplication kan hente data fra BackendApi.

Dette er en minimal implementasjon av OAuth Code Flow
